# JavaScript - Select HTML elements

## &#9758;innerHTML

會取代掉原 HTML 的內容，回傳結果是字串。

```javascript
HTMLElementObject.innerHTML = text;
```

## &#9758;innerText

會取代掉原 HTML 的內容，回傳結果是包含所有子元素之 tag。

###### 不支援 IE9 以前之瀏覽器。

```javascript
node.innerText = text;
```
